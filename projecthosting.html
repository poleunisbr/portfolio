Project Hosting

Bij Project Hosting kregen we de opdracht om een hosting platform te maken dat gericht was op studenten. Dit project loopt nog steeds.

Bij dit project komt van alle kennis dat we geleerd hebben over het hele jaar van pas. Voor ons hostingplatform gaan we een website maken waar gebruikers op kunnen aanmelden, een api die in de achtergrond alles regelt en een database om alles bij te houden. We twijfelen nog over hoe we dit volledig gaan aanpakken, maar we zijn verschillende mogelijkheden aan het testen.

Mijn Rol

Bij dit project kom ik overal van toepassing ik heb niet specifiek 1 taak maar ik spring in waar mijn team hulp nodig heeft, ik heb al mee geholpen aan de website en de api. Op een andere gelegenheid heb ik al mee geholpen aan het testen van een kubernetes cluster op de servers in het datacenter die dan het hostingplatform zou gaan draaien en ook aan andere zaken heb ik nog geholpen. Nog eens kort samengevat ik kom wel aan bod bij elk deel van het project.

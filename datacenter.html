Datacenter

Tijdens Datacenter technologie hebben wij de opdracht gekregen om een server stack op te stellen waar we ons hosting platform van Project Hosting op konden zetten.

We moesten onze servers zelf voorbereiden om ze daarna in te bouwen in het datacenter we moesten de hardware installeren onder andere: RAM, hard drives, netwerk kaarten, ... ook moesten we een OS (besturingsysteem) installeren op de servers en tot slot de servers inbouwen in de racks in het datacenter.

Mijn Rol

Tijdens dit project heb ik aan elk deel wel iets gedaan bij het installeren van de hardware heb ik de 3 servers die we hadden nagekeken en een paar hard drives, een netwerk-kaart en wat ram geïnstalleerd. De besturingssystemen van de servers heb ik allemaal geïnstalleerd. De systemen die we gebruikt hebben zijn vmware esxi, truenas en pfsense op de server die we als router gebruiken. Het inbouwen van het datacenter heb ik samen met nog 1 iemand van mijn team gedaan.
